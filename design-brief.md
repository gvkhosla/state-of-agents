# Design Brief — State of Agents

**Emotional destination:** Calm authority. The page should feel like a serious field report from someone who has done the work — not a hype launch, not a SaaS funnel.

**Visual inspiration:** pi.dev-style editorial dark mode: deep canvas, serif-led typography, monospaced labels, terminal/document motifs, rust accent, restrained blue secondary highlight, generous whitespace.

**Typography:**
- Headings/body: system serif stack (`Charter`, `Iowan Old Style`, `Georgia`) for research/editorial credibility.
- Labels/actions: system monospace for technical precision.

**Palette:**
- Deep background: `#10151C`
- Near-black depth: `#090D12`
- Text: `#EEE8DE`
- Muted copy: `rgba(220,217,211,.78)`
- Rust accent: `#E86A52`
- Deep rust: `#8F3222`
- Secondary blue: `#6A9FCC`

**Design rules:**
1. Make the report feel like the product: serious, concise, evidence-driven.
2. Use cards sparingly; keep borders thin and surfaces translucent.
3. Use micro-interactions only for feedback: button press, form submit, copy citation.
4. Avoid generic startup gradients, stock illustrations, and big animated agent mascots.
